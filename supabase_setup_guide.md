# 🚀 Panduan Lengkap Integration Supabase Backend (TWOSTRYVE)

Dokumen ini adalah panduan **langkah-demi-langkah dari A sampai Z** untuk membuat database backend gratis di **Supabase**, mengatur tabel & file storage, dan mengintegrasikannya langsung ke website & CMS **TWOSTRYVE**.

---

## 📋 Langkah 1: Buat Akun & Project di Supabase (Gratis)

1. Buka website **[https://supabase.com](https://supabase.com)**.
2. Klik tombol **"Start your project"** atau **"Sign Up"**. Anda bisa login cepat menggunakan akun **GitHub** atau Email.
3. Setelah masuk ke Dashboard Supabase, klik **"New Project"**.
4. Isi form pembuatan project:
   - **Name:** `twostryve-ecommerce`
   - **Database Password:** Buat password yang kuat (misal: `TwoStryve2026!Secret`) — *simpan password ini!*
   - **Region:** Pilih `Singapore (ap-southeast-1)` (terdekat & tercepat dari Indonesia).
   - **Pricing Plan:** Pilih **Free Tier ($0/month)**.
5. Klik **"Create new project"** dan tunggu 1-2 menit hingga database selesai disiapkan.

## ⚡ Langkah 2: Jalankan Script SQL Supabase Baru (Super Mudah & Lengkap)

1. Di menu sebelah kiri Dashboard Supabase, klik ikon **`SQL Editor`** (gambar `>_`).
2. Klik **"New Query"**.

);

CREATE TABLE IF NOT EXISTS products (

);

    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,

    status VARCHAR(50) DEFAULT 'new',
    items JSONB NOT NULL DEFAULT '[]'::jsonb,
    notes TEXT NULL,
    created_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP
);

-- 6. TABEL SETTINGS (Pengaturan Toko & Informasi Kontak)
CREATE TABLE IF NOT EXISTS settings (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    key VARCHAR(100) UNIQUE NOT NULL,
    value JSONB NOT NULL,
    updated_at TIMESTAMP WITH TIME ZONE DEFAULT CURRENT_TIMESTAMP
);

-- ==============================================================================

-- ==============================================================================

## 🔑 Langkah 3: Ambil URL & ANON KEY Supabase

1. Di sidebar Supabase, klik ikon **`Project Settings`** (gambar roda gigi/gear) di paling bawah.
2. Pilih menu **`API`**.
3. Di sana Anda akan melihat 2 nilai penting:
   - **Project URL**: (contoh: `https://xyzabc123.supabase.co`)
   - **anon / public key**: (kunci acak yang diawali `eyJhbGciOi...`)
4. **Copy kedua nilai tersebut!**

---

## 🔌 Langkah 4: Masukkan Key ke CMS Admin TWOSTRYVE

1. Buka CMS Admin Anda di browser: **[http://localhost:8090/admin.html](http://localhost:8090/admin.html)**
2. Pilih menu **`Pengaturan Toko`** di sidebar kiri.
3. Pada bagian **"Integrasi Supabase Database"**:
   - Paste **Supabase Project URL**.
   - Paste **Supabase Anon Key**.
4. Klik **"Hubungkan ke Supabase"**.
5. Selesai! Sekarang CMS dan Website TWOSTRYVE Anda sudah **100% terhubung ke database online Supabase**!
